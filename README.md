# AI-Mess-Feedback-System
An AI-driven Hostel Mess Management system with multi-lingual support, nutrition tracking, and waste estimation.
## 🚀 Features
AI Sentiment Analysis: Automatically classifies student feedback as Positive/Negative.

Multi-lingual UI: Supports 31+ languages including Hindi, Telugu, and Tamil.

Waste Predictor: Startup-ready logic to estimate food waste in kg.

Nutrition Tracker: Real-time macro-calculation (Protein/Calories).

AI Suggestion Engine: Provides actionable business insights for managers.

## 🛠️ Tech Stack
Frontend: Streamlit (Python)

AI Models: Hugging Face Transformers (DistilBERT, BART)

Data: Pandas & Plotly

## 🏃 How to Run
Clone the repo: git clone <your-repo-link>

Install dependencies: pip install -r requirements.txt

Run the app: streamlit run app.py
